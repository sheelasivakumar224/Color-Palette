# Make your own color palette 

A simple web application that allows users to generate color palettes by shuffling colors and locking selected colors while generating new ones.

### Technologies used:

- HTML5
- CSS
- JavaScript (String concatenation, loops, functions, Math functions)

### Flow:

- Colors are randomly generated using JavaScript Math functions.
- Users can lock selected colors.
- Only unlocked colors change when the palette is shuffled.
- Users can copy the hex value of the color.


---

This project was built to practice core JavaScript concepts and basic DOM manipulation while creating a functional and interactive web application.
